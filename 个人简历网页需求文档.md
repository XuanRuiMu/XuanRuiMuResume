# 个人简历网页需求文档

## 1. 项目概述

### 1.1 目标

制作一个以 3D 剧场为载体的在线个人简历网站，用于展示个人在 IT、教育、设计、音乐、娱乐与其他五个方向的能力。网站需要在 Edge、Chrome 等现代浏览器上呈现高质量的 3D 交互体验，突出技术能力与创意设计能力。

### 1.2 核心定位

- **主线**：IT 技术能力（Java 后端 / AI 应用 / 全栈开发）是核心。
- **支线**：教育、设计、音乐、娱乐与其他四个方向并行展示。
- **气质**：冷静、专业、可信，同时具有科技感和艺术感。

### 1.3 目标受众

- 校招/实习面试官、HR
- 潜在合作方或客户
- 对个人能力感兴趣的用户

## 2. 设计方向：剧场中的五个 3D 实体意向

### 2.1 内核

“剧场是我的能力舞台，五个 3D 实体意向分别代表 IT、教育、设计、音乐、娱乐与其他五个方向。Hover 时它们苏醒，点击时它们展开成各自的空间。”

### 2.2 空间结构

- **剧场内部**：昏暗的室内剧场空间，无远景，聚焦舞台。
- **中心观测台**：用户初始位置，可 360° 环视五个 3D 实体意向。
- **五个 3D 实体意向环形排列**：以 72° 间隔围绕中心，每个实体对应一个方向。
- **每个实体都是一处可交互装置**：静止时悬浮微动，Hover 时苏醒，点击时展开/变形，进入后展示内容。

### 2.3 统一交互规范

| 阶段 | 触发 | 动画描述 | 时长 |
| --- | --- | --- | --- |
| 静止 | 无交互 | 实体悬浮/缓慢自转，表面透出主题色微光 | — |
| Hover 唤醒 | 鼠标移入 | 实体光晕增强，内部结构预演，轻微靠近用户 | 0.3s |
| 点击展开 | 鼠标点击 | 实体分解/展开/变形，伴随追光亮起与粒子效果 | 0.6–0.8s |
| 推入场景 | 与展开同步 | 摄像机沿实体中轴平滑推入其内部空间 | 0.8s |
| 内容展示 | 进入完成后 | 右侧内容面板滑出，场景完全展开 | 0.4s |
| 返回剧场 | 点击返回 | 实体反向收拢/还原，摄像机退回中心 | 0.7s |

**单次进入总时长控制在 1 秒以内。**

## 3. 五个 3D 实体意向详细设计

### 3.1 IT — Agent 节点网络

- **3D 实体意向**：悬浮的 Agent 节点集群 / 数据神经网络雕塑。
- **主题色**：冷青色 #00D9FF + 深空蓝 #0A1628
- **静止状态**：实体缓慢自转，表面节点图标微弱闪烁，透出数据流光。
- **Hover 唤醒**：节点图标自连扩展成网状拓扑，实体内部可见流动的光河。
- **点击/进入**：实体分解为数据隧道，粒子节点向两侧掠过，镜头进入 Agent 节点网络空间。
- **进入后内容**：
  - 右侧滑出深色玻璃面板（占屏 35%）。
  - 顶部：动态 Agent 网络拓扑图。
  - 中部：Agent 能力卡片（Planner / Coder / Reviewer / Debugger 等），可点击展开。
  - 底部：项目列表（暮澜纪元服务端、AI 自动化脚本、MCP 工具链等）。

### 3.2 教育 — 知识/教学空间

- **3D 实体意向**：悬浮的知识晶体 / 书卷与原子模型融合体。
- **主题色**：暖橙色 #FF9F43 + 象牙白 #F5F0E6
- **静止状态**：实体悬浮，表面透出暖光，铭牌为书本/原子图标。
- **Hover 唤醒**：实体表面浮现知识卡片层叠预览，公式与代码若隐若现。
- **点击/进入**：实体像翻开的书页向两侧展开，进入悬浮教学舱/知识星球。
- **进入后内容**：
  - 右侧滑出画轴式面板（占屏 35%）。
  - 顶部：B站 UP 主“玄锐暮”频道介绍与数据。
  - 中部：课程卡片（Python、C 语言、计算机网络、论文指导等）。
  - 底部：教学经历（一对一辅导、毕业论文指导）。

### 3.3 设计 — 不可能几何空间

- **3D 实体意向**：悬浮的彭罗斯三角 / 不可能几何雕塑。
- **主题色**：紫金色 #A55EEA + 炭黑 #1A1A1A + 金色点缀 #C9A227
- **静止状态**：实体悬浮，表面透出紫金色光，铭牌为不可能三角/彭罗斯阶梯图标。
- **Hover 唤醒**：实体上的几何体开始缓慢错位旋转，黑白纹理产生流动感。
- **点击/进入**：实体展开成彭罗斯阶梯空间，镜头沿悖论空间推进，黑白渐变为彩色。
- **进入后内容**：
  - 右侧滑出画轴式面板（占屏 40%）。
  - 顶部：设计作品集缩略图网格。
  - 中部：选中作品的放大预览与设计说明。
  - 底部：设计工具/风格标签（Figma、UI/UX、3D 视觉）。

### 3.4 音乐 — 全息鼓组 + Launchpad 网格

- **3D 实体意向**：悬浮的全息鼓组 + 8×8 Launchpad 网格融合体。
- **主题色**：霓虹紫 #A855F7 + 荧光粉 #FF006E + 深黑
- **静止状态**：实体悬浮，表面透出霓虹绿光，铭牌为音波/鼓垫图标。
- **Hover 唤醒**：实体浮现全息鼓组与 8×8 Launchpad 网格的微弱闪烁。
- **点击/进入**：实体化作声波震开，进入悬浮乐器空间。
- **进入后内容**：
  - 右侧滑出节拍器式面板（占屏 30%）。
  - 顶部：作品/曲目列表（带播放按钮）。
  - 中部：可视化频谱 + 节奏波形。
  - 底部：可交互的迷你 Launchpad（点击发声/光效）。

### 3.5 娱乐与其他 — 多媒体创作工坊

- **3D 实体意向**：悬浮的多媒体创作核心 / 胶片、画笔与书卷融合体。
- **主题色**：玫红色 #FF6B9D + 薄荷绿 #4ECDC4 + 奶油白
- **静止状态**：实体悬浮，表面透出玫红色光，铭牌为胶片/画笔/书卷混合图标。
- **Hover 唤醒**：实体表面闪过视频、图像、文字等多媒体碎片。
- **点击/进入**：实体像导片仓一样旋转展开，进入综合创作舱。
- **进入后内容**：
  - 右侧滑出工坊木架式面板（占屏 38%）。
  - 顶部：项目分类标签（写作 / 相声 / 游戏 / 视频）。
  - 中部：项目卡片瀑布流。
  - 底部：创作工具链 / 时间线。

## 5. 视觉与交互要求

### 5.1 整体视觉

- 暗色主背景：#0B0C15
- 低饱和环境 + 高饱和主题色点缀
- 材质：哑光金属、全息玻璃、古老剧场的丝绒残影
- 灯光：每个场景有一束微弱顶部追光，Hover 时增强

### 5.2 动画节奏

- 所有动画采用 ease-out / cubic-bezier(0.22, 1, 0.36, 1) 等柔和缓动
- 避免突兀闪烁或过长等待
- 重点突出“实体展开/变形那一瞬间”的仪式感

### 5.3 声音反馈（可选增强）

- Hover 时播放极短环境音（低频共鸣）
- 点击时播放“装置启动 + 实体展开 + 空间展开”音效
- 音乐场景可加入可交互鼓组/Launchpad 声音

## 6. 技术要求

### 6.1 技术栈

- **3D 渲染**：Three.js + React Three Fiber（R3F）
- **前端框架**：React 19 + Vite
- **动画**：GSAP 用于镜头运动和 UI 动效
- **样式**：Tailwind CSS
- **状态管理**：Zustand 或 React Context

### 6.2 性能要求

- 目标帧率：桌面端 60 FPS
- 移动端：降级为静态展示 + 简化交互
- 加载优化：模型压缩、纹理懒加载、按需初始化场景

### 6.3 浏览器兼容

- 优先支持：Edge、Chrome、Firefox、Safari 最新版
- 需要 WebGL 2 支持
- 对不支持 WebGL 的浏览器提供静态回退页面

## 8. 实现阶段建议

1. **第一阶段：3D 场景原型**
   - 搭建基础剧场空间
   - 实现五个 3D 实体意向的几何体与基础材质
   - 实现 Hover/Click 交互与镜头移动

2. **第二阶段：内容面板**
   - 实现右侧内容面板布局
   - 填充每个方向的基础内容

3. **第三阶段：视觉打磨**
   - 添加灯光、后期处理（bloom、雾效）
   - 细化每个场景的动画效果

4. **第四阶段：内容与优化**
   - 填充真实项目与经历
   - 性能优化与移动端适配
   - SEO 与静态回退
